AntiSensitive

Key Features:

  1. Customizable Erasure Process
  2. Simple and Easy to Use
  3. Customizable UI Font
  4. No Advertisement

How It Works:

  1. Select an erasing method in [Setting] page
  2. Select the storage(usually only one)
  3. Click [Go] to start
  4. Wait for completion
  5. Click [Done] to clean all data generated or [GoOn] to keep the data and continue

How to customize a new erasing method:

  1. Goto [Setting] page
  2. Click [Customize] item
  3. Customize method:
    • click one of four to add steps, max 37.
      • 00 - for writting zeros
      • FF - for writting ones
      • Dice - for writting random data stream
      • Customized value - for writting the values you assigned(4 bytes max in hex format)
    • drag left/right to delete one
    • click [trash] icon to clear all steps
    • If the steps you specify are the same as some well-known algorithms, a firework will appear with the name of the corresponding algorithm.
  4. Save it with a given name
